How to keep you disk space "clean"

Introduction
Nowadays although the price of storage disks have become significantly lower, at the same time the size of the files keeps on increasing.
Managing disk space is essential for maintaining the performance and efficiency of your devices. Over time, your laptop, mobile devices, and other computing devices can accumulate unnecessary files and data, which can lead to storage constraints and sluggish performance.
What you should already know
Before diving into disk space management, it is helpful to have a basic understanding of your operating system and the file system used on your devices. Additionally, knowing how to navigate through the device's file explorer or terminal will be beneficial.
Laptops

Laptops tend to become the devices that are managed is terms of storage.Most people take the large available storage as granted only to realize when the clutter starts hogging down the device.

Windows

Let's start with the OS which is most commonly used.
There are quite few available utilities for cleaning up un-important files.

Step 1: Erase Temporary junk

  • Disk Cleanup

    If you don't mind an old-school interface, the classic Disk Cleanup tool gets the cleaning job done. Access it by typing Disk Cleanup into the Start menu and selecting the disk you want to clean.

    After a moment, you'll see a list of unnecessary files that Windows can clean up. Assuming you're an administrator, you should first click the Clean up system files button to scan again for additional types of data.

    you can pretty much delete all the files shown here specially cleaning up old windows version can relieve a lot of load on you drive. Previous Windows installation(s)If you've recently updated to a new major version of Windows, you'll see a entry.

    Deleting this will remove the old files that Windows keeps for 10 days‚Äîthose allow you to easily go back to an earlier version. You should also avoid removing the Windows ESD installation files option, as this is used for resetting your PC via Settings.

  • Storage Sense

    Windows 10 and Windows 11 both include much of this same functionality of Disk Cleanup, with a nicer interface, at Settings > System > Storage. You can use the Storage Sense feature found there to automatically clean up files when your disk space is low, or every so often. Click Configure Storage Sense or run it now(on Windows 10) to change these options.

    In the list of what's using space on your PC, click Temporary files and you can delete the same types of data as in Disk Cleanup. Show more categories provides more info about types of data that aren't using as much space.

Step 2: Delete Large File

Now that the unnecessary files are gone it's time to clear up files which are eating up major chunks your space but you don't really need them. Yes, i am talking about the large files (i know you have at least a 100GB of "movies" there üòè)

  • Big Files

    To help us with this, download TreeSize Free, which will scan you storage and show you where the biggest files are.

    Once installed launch it using the (Administrator) so that you can scan everywhere. To start,Select Directory in the top-left and select the storage you want to free up.

    After a few moments, you'll see a visual breakdown of your PC's storage, with the largest files at the top. Click the dropdown arrow on any folder to expand by one level.

    The largest folders will probably be Users, Windows, and Program Files (and/or Program Files (x86)). T here's not much you can do in the Windows directory without causing issues (be aware of Windows folders you should never touch), so let's look at the others.

  • Uninstall Big Programs

    Under Program Files (and/or Program Files (x86) on 64-bit Windows), you'll find files for most of the apps you have installed. If you don't use some of these anymore, head to Settings > Apps > Apps & features (on Windows 11, Settings > Apps > Installed apps) to uninstall them.

    After the big chunks are gone you might want to go a little further and clean the useless bloatware that has come with windows itself. see for apps that you never use and remove those.

This concludes our discussion for windows.. most operating systems will have most of these in common and almost all these steps should be similar. Instead lets look what other OS has extra things that we can clean and some methods that are different from the above.

  • Disk Usage Analyzer(GUI)

    OK OK!, stop!, I know GUI is bloat but for be a little lenient on the people who are just starting on their linux journey.

  • ncdu(Terminal-based Disk Usage Analyzer)

    See, I got one for terminal dwellers as well. ncdu is a powerful terminal-based disk usage analyzer that allow you to check disk usage from the command line. To install ncdu use you package manager to search for ncdu.

    • install ncdu(for debian based systems)

      Sudo apt-get install ncdu

    • Usage: to analyze simply run:

      ncdu

    • Step 2: Removing unnecessary packages

  • For debian based systems(apt-get)

    In linux, package managers cache all the downloads and updates. Overtime this can lead to a massive pile.

    • Update the package and remove orphaned packages:

      sudo apt-get update
      sudo apt-get autoremove

    • Clean up cached packages

      sudo apt-get clean
